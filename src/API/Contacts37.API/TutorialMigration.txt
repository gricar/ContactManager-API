= Contact37DbContext ----------------------
Procedimento
  Entrar na pasta da API
  Apontar o Default project para a API

Criar o Migration inicial
  dotnet ef migrations add InitialCreate --project ..\..\Infrastructure\Contact37.Persistence --startup-project . --context Contact37DbContext --output-dir Data/Migrations/Contact37DbContext 
Atualizar o Banco de Dados
  dotnet ef database update --project ..\..\Infrastructure\Contact37.Persistence --startup-project .
